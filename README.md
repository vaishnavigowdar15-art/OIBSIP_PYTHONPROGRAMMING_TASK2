Features:
 Takes user input for weight (kg) and height (meters).
  Calculates BMI using the standard formula: weight / height².
  Displays the BMI value rounded to two decimal places.
  Classifies BMI into Underweight, Normal weight, Overweight, or Obese.
  Simple command-line interaction for quick health assessment.

Tools:
 Python programming language
 Built-in `input()` function for user input
 Arithmetic operations for BMI calculation
 `if-elif-else` conditional statements for classification
 `print()` function for displaying results
